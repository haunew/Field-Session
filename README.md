# Field Session / 外勤会话留痕

A personal field-work memo app — record, trace and export your on-site activities with ease.

个人外勤备忘录应用 —— 轻松记录、追溯和导出外勤活动。

---

## Features / 功能概览

| Feature | 功能 | Description |
|---------|------|-------------|
| Session Control | 会话开关 | Start/stop a field session to begin/end data collection | 
| Timeline | 时间线 | Low-frequency location tracking + user events |
| Photo & Notes | 拍照与备注 | Capture photos and add text notes on site |
| Expense Items | 消费条目 | Log expenses during field work |
| Report Export | 报表导出 | Export personal field reports for review |
